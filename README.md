# Sports-Blog-Web-App

## General Info

- This is a sports blog where users can create an account to make posts. Users can log in to their accounts to create, edit, or delete their posts.
  
## Tools

- Python
- Flask
- SQLAlchemy
- HTML/CSS
- Bootstrap

## Usage
- Install the dependencies in requirements.txt and run app.py. Use the link provided to access the web application.
